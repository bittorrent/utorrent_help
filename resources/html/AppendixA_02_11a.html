DNA

Bittorrent DNA is a content delivery system that reduces publisher server load by using Bittorrent P2P deliver. It is used only for content that you request.  You can find more  details over here.

Enable DNA powered downloads: this enables the DNA functionality in uTorrent. 

Limit DNA uploads to: you can choose to limit the amount of upload you are willing to allow to other users 

DNA Options

  • Alert me when DNA-powered downloads are added to uTorrent: Check this to be notified when a DNA download starts.
  • Disable Screen-Saver while streaming content with DNA: allows you to specify not to turn the screen off while you intend to watch and stream DNA  content.
