Playback

Playback allows you to seamlessly play your audio/video files with your default player based on file extension or specify a specific player to use for all files.

Playback Player Selection


  • Choose your default player allows you to specify which player to be used for all files for playback. You can select one of the automatically setup players in the list or browse for your own player. Use %1 to specify the filename parameter in the command line (Required).
